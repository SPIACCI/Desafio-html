LISTA DE TAGS

O que são tags?

As tags são comandos de html que efetuam certas ações dentro de um código.

  • a- Tag para ancorar links;
  • h1 - Cabeçalho da página;
  • ul - Cria uma lista não ordenada;
  • ol - Cria uma lista ordenada;
  • li - Cria uma lista;
  • h2 - Subtitulo;
  • body - Corpo do texto;
  • hr - Traça uma linha na página;
  • small - Diminui a letra;
  • head - Cria o cabeçalho do texto;
  • i - Deixa o texto em itálico;
  • strong- Deixa o texto em negrito;
  • input - Cria algo;
  • img width - Determina o tamanho da imagem;
  • blockquote - Cria uma citação no texto.

LISTA DE ATRIBUTOS

O que são atributos?

atributos são complementos das tags podendo mudar a forma, cor e tipo das tags

  • class - Atribui uma classe ao elemento;
  • id - Atribui um id ao elemento;
  • style - Permite incluir elementos do CSS;
  • lang - Define o idioma principal do elemento;
  • title - Define o título do elemento;
  • alt - Define um texto alternativo;
  • hidden - Oculta um elemento;
  • align - Permite definir o padrão de alinhamento do elemento;
  • width - Define uma largura para o elemento;
  • height - Define uma altura para o elemento.
